Документ «Поступление продукции из транспортировки»

 

 Для регистрации факта поступления продукции ФК при перевозке между удалёнными территориальными площадками следует оформлять документ «Поступление продукции из транспортировки». Документ «Поступление продукции из транспортировки» находится в разделе «Хранение и транспортировка» команда «Поступление продукции из транспортировки», откроется журнал документов «Поступление продукции из транспортировки» см. Рис. 27.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Для создания документа необходимо нажать кнопку «Создать» в журнале «Поступление продукции из транспортировки» или заполнить автоматически на основании документа «Транспортировка продукции», тогда документ полностью автоматически заполняется данными документа «Транспортировка продукции» - кнопка «Создать на основании», выбрать «Транспортировка продукции», см. Рис. 28.

Форма документа имеет форму, представленную на Рис. 29.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Реквизитный состав документа и порядок заполнения.

В шапке документа заполняется следующие реквизиты: 

Номер - указывается номер документа в системе, подставляется автоматически; 

Дата документа – указывается дата регистрации документа; 

Организация – организация от имени которой, оформляется данный документ; 

Документ отгрузки – указывается документ «Транспортировка продукции». Выбор из справочника «Транспортировка продукции»; 

Транспортное средство – указывается тип, гос. номер транспортного средства, используемого для транспортировки продукции ФК, выбирается из списка справочника «Транспортные средства». Порядок создания описан в разделе Справочник «Транспортные средства». 

Грузоперевозчик – указывается организация, осуществляющая транспортировку партий продукции ФК, выбирается из списка справочника «Контрагенты»;

Склад отправитель – указывается склад отравителя продукции ФК;

Склад получатель – указывается склад получателя продукции ФК; 

  

Табличная часть документа: 

N – номер в табличной части по порядку; 

Партия продукции - номер партии продукции ФК; 

Транспортировочное место - шаблон идентификатор транспортировочного места, выбирается из справочника списка табличной части кнопкой «Выбрать»; 

Вес брутто влажный, кг - вес влажной продукции с контейнером, тарой, упаковкой, сумма весов перевозимого, используемого продукта и сопутствующей ему упаковки ФК с ББ, кг, подставляется автоматически при выборе транспортировочного места; 

Номер пломбы - номер пломбы ББ, подставляется автоматически при выборе транспортировочного места; 

Номер места в партии – номер места в партии продукции ФК в транспортировочном месте, подставляется автоматически при выборе транспортировочного места; 

Выявленное нарушение – указывается нарушение МКР в результате транспортировки, обнаруженное при получении продукции. Выбор из справочника «Виды нарушений при транспортировке»;

Отметка о получении – указывается флаг, подтверждение факта получения продукции; 

Причина отказа – указывается причина по которой отказано в получении продукции получателем, выбирается из справочника формы «Причины отказа в приёме продукции»; 

   

Ответственный - пользователь системы, создавший документ «внутреннее перемещение продукции». 

Комментарий - комментарий к документу, заполняется при необходимости; 

 

В открывшейся форме заполнить реквизиты: Организация, Документ отгрузки, Транспортное средство, Грузоперевозчик, Склад отправитель, Склад получатель. 


 

Процедура оформления документа

 

             При оформлении документа «Поступление продукции из транспортировки» сначала заполняются поля шапки документа и только затем следует приступить к заполнению табличной части.

 

Табличная часть документа

 

В табличной части требуется указание в каждой строке «Партии продукции», «Транспортировочного места». 

Способ 1:  

1. Заполняется реквизит «Партия продукции». Кнопка «Добавить» позволяет выбрать, сохранённую ранее «Партию продукции» из списка справочника таблицы.

2. В поле «Транспортировочное место» указывается сохранённое ранее «Транспортировочное место». К выбору предоставляются только те «Транспортировочные места», которые относятся к выбранной ранее «Партии продукции», см. Рис. 30.

 

 

 

 

 

 

 

 

 

При выборе «Транспортировочного места» значения реквизитов «Вес брутто влажный, кг», «Номер пломбы», «Номер места в партии» подставляются автоматически. Если перевыбрать «Партию продукции», то поле «Транспортировочные места» автоматически очищается. 

Способ 2:

Кнопка «Подбор из документа транспортировки» выводит окно подбора транспортировочных мест из партии продукции и переноса их в документ. В окне подбора есть возможность установить быстрый отбор по партии продукции. В списке выбора транспортных мест по номеру партии продукции, тогда будут отобраны все «Транспортные места» входящие в данную партию. см. Рис. 31.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Кнопка «Перенести в документ». Перенос строк в табличную часть подбора осуществляется либо двойным кликом мыши по соответствующей строке табличной части, либо клавишей на клавиатуре «Enter», см. Рис. 32. 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

После заполнения полей и табличной части документ следует записать – кнопка «Записать» и провести – кнопка «Провести» или «Провести и закрыть».
