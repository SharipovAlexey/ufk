Документ «Внутреннее перемещение продукции»

 

Для регистрации внутреннего перемещения партий продукции ФК внутри одной территориальной площадки, предназначен документ «Внутреннее перемещение продукции», при необходимости перевозки продукции между удалёнными территориальными площадками для транспортировки, переработки или временного хранения следует оформлять документ «Транспортировка продукции» и «Поступление продукции из транспортировки», работа данных документов будет рассмотрена в соответствующих разделах ниже.

Документ «Внутреннее перемещение продукции» находится в разделе «Хранение и транспортировка» команда «Внутреннее перемещение продукции», откроется журнал документов «Внутреннее перемещение продукции» см. Рис. 12.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Для создания документа необходимо нажать кнопку «Создать» в журнале «Внутреннее перемещение продукции». Форма документа «Внутреннее перемещение продукции» см. Рис. 13.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Реквизитный состав документа и порядок заполнения.

 

Номер - указывается номер в системе;

Дата документа – указывается дата регистрации документа;

Организация – организация от имени которой, оформляется данный документ;

Склад отправитель – указывается склад отгрузки продукции после производства;

Склад получатель – указывается склад получателя продукции;

 

Табличная часть документа:

№ места - номер места по порядку;

Партия продукции - номер партии продукции;

Транспортировочное место - шаблон идентификатора транспортировочного места, выбирается из справочника «Транспортировочные места» кнопкой «Выбрать» или при необходимости создаётся новый кнопкой «Создать»;

Вес брутто влажный, кг - вес влажной продукции с контейнером, тарой, упаковкой, сумма весов перевозимого, используемого продукта и сопутствующей ему упаковки ФК с ББ, кг;

Вес нетто влажный, кг - вес влажной продукции без упаковки, ФК без ББ, кг;

Номер пломбы - номер пломбы ББ;

 

Виза отправителя - подпись должностного лица, отпустившего продукцию;

Ответственный - пользователь системы, создавший документ «внутреннее перемещение продукции».

 

 

Процедура оформления документа

 

Оформление документа «Внутреннее перемещение» начинается с заполнения поля шапки документа и только затем следует приступить к оформлению реквизитов табличной части. В табличной части требуется указание в каждой строке «Транспортировочного места».

 

Способ 1:

1.      Заполняется реквизит «Партия продукции». Кнопка «Добавить» позволяет выбрать, сохранённую ранее «Партию продукции» из списка справочника.

2.      В поле «Транспортировочное место» указывается сохранённое ранее «Транспортировочное место». К выбору предоставляются только те «Транспортировочные места», которые относятся к выбранной ранее «Партии продукции», см. Рис. 14. 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

При выборе «Транспортировочного места» значения реквизитов «Вес брутто влажный, кг», «Вес нетто влажный, кг», «Номер пломбы» подставляются автоматически. Если перевыбрать «Партию продукции», то поле «Транспортировочные места» автоматически очищается.

             

Способ 2:

             Кнопка «Подбор» выводит окно подбора транспортировочных мест из партии продукции и переноса их в документ. В окне подбора есть возможность установить быстрый отбор:

1.      По партии продукции. В списке выбора транспортных мест по номеру партии продукции, тогда будут отобраны все «Транспортные места» входящие в данную партию. см. Рис. 15.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

По наличию остатков на складе, установив переключатель в положение либо «Все», либо «Только остатки», см. Рис. 16.

 

 

 

 

 

 

 

 

 

 

 

Кнопка «Перенести в документ». Перенос строк в табличную часть подбора осуществляется либо двойным кликом мыши по соответствующей строке табличной части, либо клавишей на клавиатуре «Enter», см. Рис. 17.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

После заполнения полей и табличной части документ следует записать – кнопка «Записать» и провести – кнопка «Провести» или «Провести и закрыть».
