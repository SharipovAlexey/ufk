Документ «Акт приёма продукции»

 

Для отражения факта получения продукции ФК за определённый период – дата, с указанием документов транспортировки на склад получателя, предназначен документ «Акт приёма продукции», который находится в разделе «Хранение и транспортировка» команда «Акт приёма продукции», откроется журнал документов «Акт приёма продукции» см. Рис. 33.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Кнопка «Создать» в журнале «Акт приёма продукции» позволяет создать новый документ «Акт приёма продукции», см. Рис. 34.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Реквизитный состав документа и порядок заполнения.

 

В шапке документа заполняется следующие реквизиты:

Номер - указывается номер документа в системе;

Дата документа – указывается дата регистрации документа;

Организация – организация от имени которой, оформляется данный документ;

Грузополучатель – указывается организация, получающая продукцию ФК;

Склад – указывается склад отгрузки продукции ФК;

Табличная часть документа имеет четыре вкладки: «Документы поступления», «Комиссия», «Дополнительная информация акта приёмки», «Дополнительная информация акта нарушений».

Первая вкладка «Документы поступления» Рис. 34:

N - номер по порядку в табличной части;

Документ поступления из транспортировки – указывается «Документ поступления из транспортировки». Выбор из списка документов «Поступление продукции из транспортировки»;

Вес продукции по документу – указывается вес продукции ФК, соответствующий документу «Поступление продукции из транспортировки», подставляется автоматически при выборе документа;

Кнопка «Добавить» позволяет выбрать ранее сохранённый документ из списка документов «Поступление продукции из транспортировки» см. Рис. 35.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Кнопка «Подобрать из поступления» позволяет выбрать двойным щелчком мыши ранее сохранённые документы из списка документов «Поступление продукции из транспортировки» без закрытия данного окна, см. Рис. 36.

 

 

 

 

 

 

 

Вторая вкладка «Комиссия» см. Рис. 37:

 

 

 

 

 

 

 

 

 

 

 

 

 

N - номер по порядку в табличной части;

Член комиссии – должностное лицо, входящее в комиссию по приёму продукции;

Сл. без. – должностное лицо службы безопасности, указывается при необходимости, установкой флага в соответствующей строке для вывода в печатную форму;

 

Кнопка «Добавить» позволяет выбрать должностное лицо из справочника «Сотрудники», см. Рис. 38.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Вкладка «Дополнительная информация акта приёмки» предназначена для ввода и раскрытия в произвольной форме информации по доставке продукции. Отображается в печатной форме документа «Акт приёмки продукции», см. Рис. 39.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Вкладка «Дополнительная информация акта нарушений» предназначена для ввода и раскрытия в произвольной форме информации по нарушениям, обнаруженным при получении продукции. Отображается в печатной форме документа «Акт выявленных нарушений» см. Рис. 40.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Комментарий - комментарий к документу, заполняется при необходимости; 

Ответственный - пользователь системы, создавший документ «внутреннее перемещение продукции». 

 

После заполнения полей и табличной части документ следует записать – кнопка «Записать» или «Записать и закрыть».
