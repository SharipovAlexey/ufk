Документ «Выпуск продукции»

 

Для регистрации факта выпуска продукции, необходимо оформить документ «Выпуск продукции». Выбрать в разделе «Производство» пункт «Выпуск продукции», откроется журнал документов «Выпуск продукции» см. Рис. 1.

 

Для создания документа необходимо нажать кнопку «Создать» в журнале «Выпуск продукции». Форма документа имеет форму, представленную на Рис. 2.

 

Реквизитный состав документа и порядок заполнения.

 

В шапке документа заполняются следующие реквизиты:

Номер - номер оформляемого документа в учётной системе. Нумерация документов производится автоматически ИС, изменять данное поле не следует.

Дата документа –дата документа, совпадает с фактической датой совершения операции выпуска;

Организация – организация от имени которой, оформляется данный документ;

Склад – указывается слад на который поступает продукция после производства;

Влажность, % – влажность ФК, %;

Партия - выбирается из списка справочника «Партии», если оформляется новая партия, то необходимо создать новый элемент в справочнике «Партии». Порядок создания описан в разделе Справочник «Партии продукции»;

Вес единицы тары, кг - вес одного ББ, кг, без ФК;

Ответственный - пользователь системы, создавший документ «выпуск продукции».

 

Табличная часть документа содержит следующую информацию:

№ места - номер места по порядку;

Транспортировочное место - шаблон идентификатора транспортировочного места, выбирается из справочника кнопкой «Выбрать» или при необходимости создаётся новый кнопкой «Создать»;

Вес брутто влажный, кг - вес влажной продукции с контейнером, тарой, упаковкой, сумма весов перевозимого, используемого продукта и сопутствующей ему упаковки ФК с ББ, кг;

Вес нетто влажный, кг - вес влажной продукции без упаковки, ФК без ББ, кг. В ИС вычисляется формулой разницы значений табличной части «Вес брутто влажный, кг» реквизита шапки документа «Вес единицы тары, кг»;

Вес нетто сухой, кг - вес сухой продукции без сопутствующей ему упаковки ФК с ББ, кг. В ИС вычисляется формулой процентного соотношения «Вес нетто влажный, кг» реквизита шапки документа «Влажность, %»;

Номер пломбы - номер пломбы ББ;

В открывшейся форме заполнить реквизиты: Организация, Склад, Партия, Влажность, %, Вес единицы тары, кг.

 

Процедура оформления документа

 

При оформлении документа «Выпуск продукции» сначала заполняются поля шапки документа и только затем следует приступить к оформлению реквизитов табличной части. Если в документе не указано значение реквизита «Влажность, %», то в журнале документов «Выпуск продукции» розовым цветом будут выделены строки документов, в которых необходимо заполнить данный реквизит, см. Рис. 3.

 

 

 

 

 

 

 

 

 

 

 

 

 

Справочник «Партии продукции»

 

В справочнике партии продукции содержится перечень всех выпущенных партий продукций ФК.

Для создания новой партии нужно создать новый элемент справочника «Партии продукции», для этого необходимо выбрать в разделе «Нормативно справочная информация» команду «Партии продукции». Откроется форма списка справочника «Партии продукции» кнопка «Создать» см. Рис. 4.

Реквизитный состав справочника «Партии продукции» и порядок заполнения.

В шапке элемента заполняется следующие реквизиты:

Номер партии - номер формируемого элемента. Нумерация производится автоматически, изменять данное поле не следует;

Номенклатура - номенклатура оформляемого элемента, значение в поле подставляется автоматически, изменять данное поле не следует;

Дата выпуска – дата выпуска партии продукции, совпадает с фактической датой формирования «Партии продукции», значение в поле подставляется автоматически ИС, изменять данное поле не следует;

Время выпуска – время выпуска партии продукции;

Организация владелец – организация от имени которой, формируется данный элемент значение в поле подставляется автоматически, изменять данное поле не следует;

Общий Вес брутто влажный, кг - вес влажной продукции всей партии с контейнером, тарой, упаковкой, сумма весов перевозимого, используемого продукта и сопутствующей ему упаковки ФК с ББ, кг;

Общий Вес нетто влажный, кг - вес влажной продукции всей партии без упаковки, ФК без ББ, кг;

 

 

Табличная часть документа

 

При оформлении табличной части требуется указание в каждой строке транспортировочного места. Каждое транспортировочное место -  уникальный элемент требуется каждый раз создавать новый элемент справочника «Транспортировочные места». Для облегчения ввода транспортировочных мест есть возможность автоматического создания транспортировочных мест.

По нажатию кнопки «Создать транспортировочные места» в открывшемся окне следует ввести количество мест в выпущенной партии. При нажатии кнопки «ОК» будет создано указанное количество строк в документе и в колонке будут выбраны вновь созданные транспортировочные места.

В созданных строках табличной части необходимо ввести «Вес брутто влажный, кг» и «Номер пломбы ББ» соответственно строкам табличной части формы, ИС автоматически рассчитает «Вес нетто влажный, кг» и «Вес нетто сухой, кг».

 

Шаблон «Транспортировочного места» в табличной части формы формируется только после проведения документа, см. Рис. 6.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Далее следует записать документ – кнопка «Записать» и провести – кнопка «Провести».  При нажатии кнопки «Провести и закрыть» выполнятся действия: «записать», «провести» и форма документа будет закрыта.

В процессе проведения документа выполняется ряд проверок и, если в процессе проведения будут обнаружены ошибки документ не будет проведён, и не будет участвовать в процессе учёта и формировании отчётности. Подготовленный документ обязательно должен быть проведён.

Причина, по которой документ не может быть проведён, выводится в нижней части формы сообщение об ошибках.

Приведём несколько примеров проверки:

-«Общий вес» продукции, указанный в «Партии» не соответствует «Весу продукции  упакованному в транспортировочные места»;

-«Общий вес нетто» продукции, указанный в партии, не соответствует «Весу нетто продукции упакованному в транспортировочные места»;

-«Дата выпуска» партии продукции отличается от даты документа, см. Рис. 7.
